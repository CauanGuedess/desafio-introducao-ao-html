Primeiros computadores

(Voltar)

Podemos dividir o surgimento dos computadores em 4 gerações.
A primeira dentre elas é onde surgiu os primeiros computadores, o Colossus (1946) e o Eniac (1946), ambos eram enormes e dependiam de muita energia para funcionar.
Na segunda geração foi onde surgiu os transitores, os quais ampliavam o sinal emitido, reduziam o tamanho dos computadores e serviam de controladores para interromper a passagem de corrente elétrica.
Já a terceira geração foi onde surgiu os circuitos integrados, foram implementados transitores, capacitores e resistores, o que fez com que o tamanho dos computadores reduzissem a um nível próximo ao que vemos hoje.
A quarta geração se inicou em 1971 e ocorre até hoje, foi marcada pela invenção do mouse, maior desempenho de computadores e surgimento de aparelhos móveis como o celular.
Muitos acreditam que existe uma QUINTA geração, a qual se refere a IA ( Inteligência Artificial ), Computadores Quânticos, Cripto Moedas, Metaverso.

O que são Clients?

(Voltar)

Clients são os dispositivos que entram em contato com o server através da internet, é quem consome os serviços de algum provedor, são geralmente representados por computadores pessoais, smartphones, tablets.

O que são Servers?

(Voltar)

Server é o que armazena e atende as informações que são requisitadas pelo Client, são capazes de executar aplicações, gerir processamentos e armazenar dados, são geralmente computadores de alta qualidade.

HTML é Linguagem de Programação?

(Voltar)

A resposta é não, HTML é a sigla para Hyper Text Markup Language, assim sendo um arquivo de marcação e não uma linguagem de programação, o HTML é um arquivo lido e interpretado pelos navegadores para que pessoas acessem e consigam ler o que neles estão escritos ou ilustrados.
